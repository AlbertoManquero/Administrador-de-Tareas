# Administrador-de-Tareas
Administrador de tareas en Python que permite detectar los procesos principales que consumen más memoria, buscar procesos individuales por id y salir.

Además el código "simulación" simula el inicio de un sistema operativo y permite trabajar con procesos.
USUARIO = Admin
CONTRASEÑA = 1234
memoria_total = 8192 mb

Se puede:
Iniciar procesos
Modificar procesos
Aumentar memoria
Ver procesos activos
Apagar sistema
